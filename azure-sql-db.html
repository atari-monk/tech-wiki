Description

Create a single database - Azure SQL Database.

Prerequisites/Assumptions

  1. I have Azure free account set up.
  2. Using a local installation of Azure CLI.

Tutorial

  1. Run PowerShell in chosen root folder.

  2. Check Azure CLI version.

    az version

  3. Update Azure CLI.

    az upgrade

  4. Update Azure CLI.

    az upgrade

  5. During Azure CLI MSI install, phyton needed to be killed.

    taskkill /im python.exe /F

  6. Sign in to the Azure CLI.

    az login

  7. Create resource group.

    details

    Choose resource group name

    location

    tag - descriptor

    az group create --name atari-monk-az-sql-rg --location swedencentral --tags create-and-configure-database

  8. Create a server.

    details

    Choose server name

    Choose resource group name

    location

    server login

    Change it to your strong password

    az sql server create --name atari-monk-az-sql-server --resource-group atari-monk-az-sql-rg --location swedencentral --admin-user atari-monk-az --admin-password password

  9. Configure a server-based firewall rule.

    details

    Choose resource group name

    Choose server name

    ip fillter, set your public ip, of pc you will use with this db

    ip fillter, set your public ip, of pc you will use with this db

    az sql server firewall-rule create --resource-group atari-monk-az-sql-rg --server atari-monk-az-sql-server -n AllowYourIp --start-ip-address start-ip --end-ip-address end-ip

  10. Deleting resources, server and dbs, after they are not needed.

    details

    Choose resource group name

    az group delete --name atari-monk-az-sql-rg
