Description

How to install c++ compiler on windows.

Tutorial

  1. Download MSYS2 cli app, link in Sources.

  2. Choose intallation folder.

    details

    Choose your own installation folder path

    C:\atari-monk\Apps\msys64

  3. skipped many steps here, for now...

  4. Open 'View Advanced System Settings' dialog in windows

    details

    Text to search config dialog in windows

    Choose your own path to compiler

    Choose your own path to make

    Advanced System Settings
    Environment Variables button
    Select Path, edit
    Add or edit path to compiler
    Add or edit path to make
    Restart all instances of powershell for envirenmental vars to go in effect

  5. Make sure VSCode make extensions have correct path to make in setting.json (acces it thrue extensions)

  6. Note on freeglut. To add it to compiler libs use bellow comand in msys2

    details

    msys2 freeglut install command

    pacman -S mingw-w64-x86_64-freeglut
