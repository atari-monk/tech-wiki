Description

Collision detection between rectangles (not rotated).

Prerequisites/Assumptions

  1. None.

Tutorial

  1. Example described below
    rect definition:
    var rect1 = { x:x1, y:y1, width: 40, height: 40 }
    var rect2 = { x:x2, y:y2, width: 40, height: 40 }
    x,y is top left point
    given theese assumptions:

    Below condition detects collison on left
    rect1.x < rect2.x + rect2.width

    Below condition detects collison on right
    rect1.x + rect1.width > rect2.x

    Below condition detects collison on top
    rect1.y < rect2.y + rect2.height

    Below condition detects collison on bottom
    rect1.y + rect1.height > rect2.y

    Less readable but more efficient version:

    rect1.x > rect2.x + rect2.width
    || rect1.x + rect1.width < rect2.x
    || rect1.y > rect2.y + rect2.height
    || rect1.y + rect1.height < rect2.y
