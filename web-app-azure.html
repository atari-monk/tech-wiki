Description

Deploy web app with db to azure.

Prerequisites/Assumptions

  1. I have Azure free account set up.
  2. Azure CLI installed locally.
  3. Azure SQL Database is done before this one.

Tutorial

  1. Open Code powershell (shell set on folder Code)

  2. Clone example repo.

    git clone https://github.com/Azure-Samples/msdocs-app-service-sqldb-dotnetcore.git
    cd msdocs-app-service-sqldb-dotnetcore

  3. Login to Azure CLI

    az login

  4. Create resource group for the project.

    details

    proj resource group name

    location

    az group create --name az-web-app-db-tutorial-rg --location swedencentral

  5. Create an App Service plan.

    details

    App Service plan name

    proj resource group name

    Free az plan tier

    az appservice plan create --name az-web-app-db-tutorial-plan --resource-group az-web-app-db-tutorial-rg --sku F1

  6. See available runtimes for your app. To choose one. (optional)

    details

    for Linux

    for Windows

    az webapp list-runtimes --os linux --output table
    az webapp list-runtimes --os windows --output table

  7. Create the App Service web app.

    details

    Generates domain name https:// .azurewebsites.com

    runtime

    az webapp create --name az-web-app-db-tutorial --runtime "DOTNET|6.0" --plan az-web-app-db-tutorial-plan --resource-group az-web-app-db-tutorial-rg

  8. Create an Azure SQL Server to host the database
    This was done in tutorial Azure SQL Database

  9. Create db on server.

    details

    rg already created in az for sql server

    server already created in az

    name of db

    az sql db create --resource-group atari-monk-az-sql-rg --server atari-monk-az-sql-server --name coreDb

  10. Connect the App to the Database.
    When prompted, provide the administrator username and password for the SQL database.
    Copy this connection string value from the output for later.

    details

    rg witch contains the web app

    web app name

    rg witch contains the sql server

    server already created in az

    name of db

    To see the entirety of the command output, drop the --query in the command

    az webapp connection create sql --resource-group az-web-app-db-tutorial-rg --name az-web-app-db-tutorial --target-resource-group atari-monk-az-sql-server --server atari-monk-az-sql-server --database coreDb --query configurations

  11. Generate the Database Schema.

  12. Set up a firewall rule on the SQL database server.
    This was done in tutorial Azure SQL Database

  13. Update the appsettings.json of app, with generated connection string.

    details

    Example connextion string

    "AZURE_SQL_CONNECTIONSTRING": "Data Source=.database.windows.net,1433;Initial Catalog=coreDb;User ID=;Password="

  14. Update app setup in program.cs (older apps may have it in Startup.cs)

    details

    set connection string name to use with ef core context and sql server

    services.AddDbContext(options => options.UseSqlServer(Configuration.GetConnectionString("AZURE_SQL_CONNECTIONSTRING")));

  15. From a local terminal applay migrations.

    details

    Assuming root "C:\atari-monk\Code"

    run migrations

    cd msdocs-app-service-sqldb-dotnetcore\DotNetCoreSqlDb
    dotnet ef database update

  16. error Client with IP address xxx.xxx.xxx.xxx is not allowed to access the server
    means your Azure firewall rule is incorrect
    use ip from error to update it.

  17. Deploy to the App Service with VS Code
    Install Azure Account and Azure App Service extensions
    Login with Account extension

    details

    deployable publish folder for the app in the bin/release/publish directory

    dotnet publish -c Release

  18. Right-click on publish folder and select Deploy to Web App.

  19. Test web app in the browser.

  20. Test web app in the browser.
    In Azure Portal, Browse link at the top of the App Service overview page

  21. Use Kudu tool to get info on deployment.

    details

    For apps deployed in Free, Shared, Basic, Standard, and Premium App Service plans

    https://.scm.azurewebsites.net

  22. Configure and Stream Application Logs.
    To filesystem or console.

    details

    web app service name

    web app service name

    proj resource group name

    az webapp log config --web-server-logging 'filesystem' --name az-web-app-db-tutorial --resource-group az-web-app-db-tutorial-rg
    az webapp log tail --name az-web-app-db-tutorial --resource-group az-web-app-db-tutorial-rg

  23. Clean up resources

    details

    proj resource group name

    az group delete --name az-web-app-db-tutorial-rg
