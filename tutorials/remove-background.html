Description

How To Remove Background Without A Green Screen In OBS and Streamlabs.

Prerequisites/Assumptions

  1. NVIDIA Broadcast
  2. Choose one of stream recorder:
    1. Streamlabs
    2. OBS Studio
    3. Bandicam
    4. GeForce Experience

Tutorial

  1. Open the Nvidia broadcast app and navigate to the Camera tab.

  2. Select your camera source

  3. Select the resolution you want the camera to be.

  4. Select effect to “Background Removal” option.

  5. Add created virtual camera to Streamlabs/OBS.
    Go to sources -> Add Source -> Camera -> Select Nvidia Broadcast.
    You can resize and position this as you like.

  6. Use Nvidia Broadcast Microphone Features to remove background noise.

  7. Instead of removing your background you can replace it with any image you want.
