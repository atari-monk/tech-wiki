Description

How to install c++ compiler on windows.

Prerequisites/Assumptions

  1. None.

Tutorial

  1. Download MSYS2 cli. Link in Source.

  2. Choose intallation folder. Finish install wizard.

    details

    Choose your own installation folder path

    C:\atari-monk\Apps\msys64

  3. In windows, search for this cli.

    MSYS2 MSYS

  4. Run cmd to update the base packages. Type 'y' on prompts. Close.

    pacman -Syu

  5. Open MSYS2 MSYS cli second time. Update packages.

    pacman -Su

  6. skipped many steps for now, update from source... to install compiler and debuger

  7. To set Environment Variables (how system finds exe), Open this dialog in windows

    details

    Text to search config dialog in windows

    View Advanced System Settings

  8. Click Environment Variables button, Path, add or edit path to compiler.

    details

    Choose your own path to compiler

    C:\atari-monk\Apps\msys64\mingw64\bin

  9. Restart all instances of powershell for envirenmental vars to go in effect

  10. Make sure VSCode make extensions have correct path to make in setting.json (acces it thrue extensions wiget)

  11. Note on freeglut. To add it to compiler libs use bellow comand in msys2

    details

    msys2 freeglut install command

    pacman -S mingw-w64-x86_64-freeglut

Summary

I use this c++ compiler with makefile generated form panda ide.
This is for my old c++ projects with glut.
I refactored archive app.
It uses freeglut.
