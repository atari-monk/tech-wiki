Description

Beginner guide to html.
Web official tag language for web pages.

Prerequisites/Assumptions

  1. HTML docs
  2. HTML tags
  3. HTML validator
  4. HTML Escape/Unescape
  5. Chrome

Tutorial

  1. Below only first time Notes from lessons. Html code in Repository(!)

  2. setup

    install chrome extension dark new tab
    install VSC extension prettier - code formatter
    install VSC extension vscode-icons
    install VSC extension github theme, set dark default
    install VSC extension live server

  3. lesson01 - template

    index.html - lower case convention for hosting reasons
    html - html page
    head - metadata on the page
    body - seen in browser
    h1 - header
    p - paragraph, write p and hit tab

  4. lesson02 - template

    meta author
    meta desc
    alt+z to Word Wrap in vscode
    icon

  5. lesson03 - text

    alt+z on new file
    shift + alt + down - copy line or many lines
    h1 title, h2 topic, h3 subtopic
    headers give hierarchy to content
    hr - line
    white space collapsing in html
    br - line break
    block element - breaks line, like p, h1
    inline element - dosnt break line like em - emphasise, strong - strong emphasise
    abbr - tooltip with expand of abbrevietion
    html entity - special chars
    &nbsp; - space
    &lt; - <
    &copy; - copy right symbol
    address - paragraph with address
    <!-- --> - comment
    ctr+u or inspect in browser shows page code so also comments
    in page code one can see live server javascript addons

  6. lesson04-lists

    ctrl+D to highlight same section below
    li - list element, type li press tab
    ol - ordered list
    ul - unordered list
    li - dosnt create extra space between lines unless p is in the li
    dl - description list
    dt - description term
    dd- description detail

  7. lesson05 - link

    a - link pages
    href - absolute text ip address for other servers
    relative path on same server, project
    lorem*1 - adds placeholder text
    section - divides page into sections with attribute id (name)
    nav - navigation, for ex. on list with a (anchors)
    # - before id name in href to section
    in anchor (a) text place shortest possible desc of destination page
    download - in lets download some resource
    mailto: - mail in href, not used since crawlers colect mails, contact forms instead
    tel:
    target='_blank' - link opens new tab
    / - in href links to default page in same folder

  8. lesson06 - image

    img tab - for img markup
    alt - text if img is not reachable
    title - text in tooltip on img
    width, height - tell browser default img size so it dosent shift it
    loading - eager is default, lazy - loads only when scrolling
    br*150 - types 150 break lines
    figure - html element used with img and figcaption element (desc)
    single ' can be used inside ""
    code - element for code display

  9. lesson07 - semantic tags

    h1 - one per page, title
    h2 - subtopic
    h1 to h3 per page
    nav
    hr - topic change
    header - header of the page
    main - marks main page content
    footer - footer of a page
    multiple nav should be labeled
    article - clear topic with h2
    can contain sections with h3
    aside
    details
    summary
    mark - highlight text
    time - with datetime attribute
    div - dividers, do not use them as they dont provide meaning to assistive technology
    span - as div, use only when no other semantic element with more meaning
    html outliner - chrome extension

  10. lesson08 - tables

    tables structure data, do not use them as page layout as in 90's
    table
    tr - table row
    td - table data
    shift alt down arrow on selected row helps
    to see table css is needed
    table expands to it's content, not to width of page
    th - table headings
    colspan, rowspan - attributes
    semantics of table
    table
    caption
    thead
    tbody
    tfoot
    scope - th attribute row or column

  11. lesson09 - input forms

    add nav section, add article, h2
    form - with action, method
    dont send sensitive info with get as they are in url

  12. lesson10 - beginner project

    every tag of html has inteli sense, just type name and press tab
