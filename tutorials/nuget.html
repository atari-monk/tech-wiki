Prerequisites/Assumptions

  1. .NET SDK
  2. nuget.org free account

Tutorial

  1. Create new github repo

  2. Open folder with VSCode

  3. Setup project.

    dotnet new sln
    dotnet new classlib -n Connection.Builder
    dotnet sln add ./Connection.Builder/

  4. In csproj file set properties describing nuget

    details

    Id name, unique in nuget.org

    version

    author

    company name

    tags important for searching nuget

    description

    <PackageId>Connection.Builder.15.11.2022.001.Test</PackageId><br><Version>1.0.0</Version><br><Authors>atari-monk</Authors><br><Company></Company><br><PackageTags>tag1;tag2</PackageTags><br><Description>Builds ConnectionString.</Description>

  5. Build

    dotnet build

  6. Produce nuget file with dotnet cli.

    details

    cd folder with csproj file

    clear your local packages cache

    clear your local packages cache

    build path

    cd C:\atari-monk\Code\connection-builder\Connection.Builder
    dotnet nuget locals all --clear
    dotnet restore
    dotnet pack -c Release -o C:\atari-monk\Build\nugets

  7. In target csproj file, add local nuget source. (also nuget.org)

    <RestoreSources>$(RestoreSources);C:\atari-monk\Build\nugets;https://api.nuget.org/v3/index.json</RestoreSources>

  8. Consume nuget in target project.

    details

    PackageId

    cd C:\atari-monk\Code\repo\proj
    dotnet add package Connection.Builder.19.11.2022
    dotnet restore
