Tutorial

  1. Create new github repo.

    identity-web-app
    cd C:\atari-monk\Code
    git clone https://github.com/atari-monk/identity-web-app.git
    cd identity-web-app
    code .

  2. Generate web app with identity

    details

    check options

    dotnet new list
    dotnet new mvc --auth Individual -uld -n identity-web-app

  3. Setup project.

    dotnet new sln
    dotnet sln add identity-web-app
    dotnet tool update --global dotnet-ef
    cd .\identity-web-app\
    dotnet ef database update

  4. Restart VSCode (after adding sln, for some reason it needs it).

    details

    add to .gitignore

    *.vscode

  5. Run the app and register a user and login. (With debugger and/or without).

    dotnet run
