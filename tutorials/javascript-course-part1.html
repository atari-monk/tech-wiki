  9. Data Types

    Value is Object or Primitive
    7 Primitives, 3 base and 4 rest
    Number - floating point numbers used for decimals and integers
    String - sequence of characters used for text
    Boolean - true or false used for decisions
    Undefined - undefined variable value (empty)
    Null
    Symbol - immutable and unique
    BigInt - larger integers than Number can hold
    dynamic typing - value has type, not variable,
    variable can be assign/reassign to any type/value,
    also undefined in type and value
    Comment - //, Ctrl+/ in VSCode, /**/
    typeof - used on variable or value
    language bug - typeof null return object (it's not)
