Description

Net development envirenment secrects are also implemented
in production environment on Azure cloud.
Add key, value pairs to define entries that will be available in
same way as local secrets in IConfiguration of app.

Prerequisites/Assumptions

  1. Azure subscription
  2. App deployed in Azure App Service

Tutorial

  1. Go to Azure portal. Browse Web app deployed in App Service.

  2. Go to Configuration, Application settings.

  3. To see details of errors on your page set production envirenment to development mode.

    details

    Value

    Development
